Getting Started

Prerequisites

  • A Nebelus API key (format: sk-ns-org-...)
  • Modern browser with microphone access
  • HTTPS or localhost (required for getUserMedia API)

Quick Start

  1. Click "Launch Demo" on any example above
  2. Enter your Nebelus API key
  3. Click "Connect" to establish WebSocket connection
  4. Click "Start Translation" and allow microphone access
  5. Speak naturally and watch real-time translation!

Local Development

# Clone the repository
git clone https://github.com/Nebelus/docs.git
# Serve vanilla JS example
cd examples && make vanilla-js
# Or serve Vue example
make vue-html
